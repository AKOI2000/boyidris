# Boyidris Portfolio Website

A personal portfolio website developed to showcase projects, skills, and professional work for a client.

The website highlights projects and provides a central place where visitors can learn about the developer and view their work.

---

## Features

- Project showcase section
- Clean portfolio layout
- Responsive design for all devices
- Reusable React components
- Fast loading and optimized UI

- CSS
